GIT PULL
To get the changes, which were made in the remote, from remote repo to local repo

Note: without pulling the changes available in remote repo, we cannot push the changes to remote repo

PULL Vs CLONE
Clone will get the complete code from remote to local
Pull will get only the changes available in remote

FORKING A REPO
